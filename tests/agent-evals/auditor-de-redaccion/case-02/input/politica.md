# Politica de respaldo y retencion de datos

## 1. Proposito
Definir como se respaldan y por cuanto tiempo se conservan los datos de clientes.

## 2. Alcance
Aplica a las bases de datos productivas y a los archivos adjuntos de los clientes.

## 3. Frecuencia de respaldo
Se realiza un respaldo completo diario y respaldos incrementales frecuentes.

## 4. Retencion
Los respaldos se conservan 90 dias y luego se eliminan de forma segura.

## 5. Cumplimiento regulatorio
Para cumplir la normativa vigente, los datos de clientes se conservan un minimo de 1 anio.

## 6. Restauracion
El equipo de operaciones puede restaurar un respaldo cuando sea necesario.

## 7. Responsables
El administrador de sistemas ejecuta y supervisa esta politica.
